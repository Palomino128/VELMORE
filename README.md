# VELMORE - Sistema Web de Catálogo de Perfumes

Proyecto académico desarrollado en Java Spring Boot para gestionar un catálogo de perfumes.

## Funcionalidades

- Inicio del sistema.
- Catálogo de perfumes.
- Búsqueda por nombre, marca, descripción o categoría.
- Detalle del producto.
- Contacto mediante mensaje para WhatsApp.
- Login administrativo.
- Panel administrativo.
- Registro, edición y eliminación de productos.
- Descarga de reporte Excel.

## Librerías implementadas

- Google Guava: caché y colecciones.
- Apache POI: reportes Excel.
- Apache Commons Lang: validación y limpieza de textos.
- Apache Commons IO: manejo de flujos para reportes.
- Logback: registro de eventos y errores.

## Seguridad

- Login de administrador.
- Rutas administrativas protegidas.
- Control de sesión.
- Logout.
- Cabeceras de seguridad.
- Validación de datos.

## Ejecución

1. Abrir en NetBeans IDE 19.
2. Ejecutar `VelmoreLibreriasApplication.java`.
3. Abrir `http://localhost:8081`.
